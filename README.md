# dillinger wrapped in electron

The fantastic [dillinger.io](http://dillinger.io) wrapped in an Electron app. Mind-blowingly simple.

To run:

    $ git clone git@github.com:nornagon/dillinger-app
    $ cd dillinger-app
    $ npm install
    $ npm start
